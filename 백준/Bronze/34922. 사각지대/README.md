### 문제 설명

<p>한별이는 오늘따라 너무 공부하기 싫어서 선생님 몰래 수업 시간에 딴짓하려 한다.</p>

<p>교실은 가로 길이가 $w$, 세로 길이가 $h$인 직사각형 모양이고, 선생님은 교실의 왼쪽 위 꼭짓점에 있다. 선생님은 업무를 하느라 매우 바쁘기 때문에 선생님과 거리가 $r$ 이하인 곳만 볼 수 있다. 선생님이 볼 수 있는 구역에서 딴짓을 하면 들키지만, 볼 수 없는 구역에서 딴짓을 하면 들키지 않는다.</p>

<p>한별이가 선생님에게 들키지 않고 딴짓할 수 있는 영역의 넓이를 구하시오.</p>

### 입력 

 <p>첫째 줄에 교실의 가로 길이 $w$와 세로 길이 $h$가 공백으로 구분되어 주어진다. $(1 \leq w, h \leq 100)$</p>

<p>둘째 줄에 선생님이 볼 수 있는 거리 $r$가 주어진다. $(1 \leq r \leq \min(w, h))$</p>

<p>입력으로 주어지는 수는 모두 정수이다.</p>

### 출력 

 <p>한별이가 선생님에게 들키지 않고 딴짓할 수 있는 영역의 넓이를 출력하시오.</p>

<p>절대/상대 오차는 $10^{-2}$까지 허용한다. 문제의 정답 $a$에 대해 출력 $x$가 $a-0.01 \leq x \leq a+0.01$ 또는 $0.99a \leq x \leq 1.01a$를 만족하면 <span class="result-ac-text"></span>를 받을 수 있다.</p>
