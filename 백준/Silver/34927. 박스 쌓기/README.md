### 문제 설명

<p>폴의 집에 $N$개의 박스가 있다. $i$번째 박스의 무게는 $w_i$이다. 폴은 박스를 하나씩 한 줄로 쌓아 탑을 만들려고 한다.</p>

<p>모든 박스는 자신 위에 쌓인 박스들의 무게 합이 자신의 무게 이하가 되게 쌓아야 한다.</p>

<p>모든 박스의 무게가 주어질 때 최대 몇 개의 박스를 쌓을 수 있을지 구해보자.</p>

### 입력 

 <p>첫째 줄에 박스의 개수를 의미하는 양의 정수 $N$이 주어진다. $(2 \le N \le 300\,000)$</p>

<p>둘째 줄에 박스의 무게를 의미하는 양의 정수 $w_1, w_2, \cdots, w_N$이 공백으로 구분되어 주어진다. $(1 \le w_i \le {10}^{9})$ </p>

### 출력 

 <p>첫째 줄에 쌓을 수 있는 박스의 최대 개수를 출력한다.</p>
