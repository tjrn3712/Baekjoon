### 문제 설명

<p>슬기로운 마이폴학교 학생들은 인터넷 연결이 되지 않을 때마다 공룡 게임을 즐긴다. 공룡 게임은 길이 $N$의 1차원 맵에서 아래 규칙으로 진행된다.</p>

<ul>
<li>$1$번 칸과 $N$번 칸을 제외한 칸에는 장애물이 놓여있을 수 있다.</li>
<li>공룡은 $1$번 칸에서 출발하며, $N$번 칸까지 장애물이 놓인 칸을 밟지 않고 도착하면 게임을 클리어한다.</li>
<li>공룡은 $N$번 칸에 도착하기 전까지 매 턴마다 다음 두 가지 행동 중 하나를 선택할 수 있다.
<ol>
<li>걷기: 공룡이 $i$번 칸에서 $i+1$번 칸으로 이동한다.</li>
<li>점프: 공룡이 $i$번 칸에서 $i+K$번 칸으로 이동한다. 단, $N$보다 큰 번호의 칸으로 이동할 수 없다.</li>
</ol>
</li>
</ul>

<p>맵의 상태와 공룡의 점프 거리 $K$가 주어질 때 게임을 클리어할 수 있는지 판별하라.</p>

### 입력 

 <p>첫째 줄에 맵의 길이를 나타내는 정수 $N$과 공룡의 점프 거리를 나타내는 정수 $K$가 공백으로 구분되어 주어진다. $(2\le K<N\le 10^6)$</p>

<p>둘째 줄에 공룡 게임의 맵을 나타내는 길이 $N$의 문자열이 주어진다. <span style="color:#e74c3c;"><code>_</code></span>는 장애물이 없는 칸, <span style="color:#e74c3c;"><code>#</code></span>은 장애물이 있는 칸을 의미한다.</p>

### 출력 

 <p>첫째 줄에 주어진 게임을 클리어할 수 있다면 <span style="color:#e74c3c;"><code>YES</code></span>, 불가능하다면 <span style="color:#e74c3c;"><code>NO</code></span>를 출력한다.</p>
